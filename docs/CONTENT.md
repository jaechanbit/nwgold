# 콘텐츠 기준

## 말투

- 따뜻하고 친절하게 설명한다.
- 전문 용어보다 고객이 평소 사용하는 말을 쓴다.
- 짧고 구체적인 문장을 사용한다.
- 구매를 재촉하거나 불안을 자극하지 않는다.
- `최고`, `무조건`, `전국 최저가`처럼 증명하기 어려운 표현을 쓰지 않는다.

## 신뢰를 표현하는 방법

추상적인 자랑보다 확인 가능한 사실을 보여준다.

- 경력은 확인된 표현인 `20년 이상의 경력`으로 구체적으로 표시한다.
- `능숙한 수리`는 수리 가능한 품목, 과정, 전후 사진으로 보여준다.
- `친절한 상담`은 상담 과정, 예상 소요 시간, 실제 고객 후기 등으로 설명한다.
- 후기는 고객의 허락을 받고 실제 후기만 사용한다.
- 금 매입 가격은 기준 시점과 품위, 중량 등 적용 조건이 명확할 때만 표시한다.

## 가격 표기

- 관리자는 모든 공개 상품의 가격을 매주 확인한다.
- 가격 확인 후 14일이 지나면 해당 상품을 공개 화면에서 자동으로 숨긴다.
- 숨겨진 상품은 삭제하지 않고 `가격 확인 필요`로 관리한다.
- 관리자가 가격을 다시 확인하고 확인일을 갱신하면 상품을 다시 공개할 수 있다.
- 모든 공개 상품에 원화 가격을 표시한다.
- 천 단위 구분을 사용해 `350,000원`처럼 쓴다.
- `가격 문의`로 대체하지 않는다. 가격을 확인하지 못한 상품은 게시하지 않는다.
- 할인 가격을 표시할 때는 적용 기간과 조건을 함께 안내한다.
- 중량이나 옵션에 따라 달라지는 가격은 기준 중량, 옵션, 추가 비용을 설명한다.
- 가격 영역 가까이에 최종 가격 확인 날짜 또는 시각을 표시한다.
- 실제 판매가와 사이트 가격이 다를 수 있는 조건이 있다면 구체적으로 고지한다.
- 단순한 면책 문구로 부정확한 가격을 방치하지 않는다.

## 핵심 메시지 초안

### 제목 후보

20년의 경험으로 정성껏 상담하는 남원금방

### 소개 문구 후보

반지와 목걸이, 예물, 순금 제품부터 금 매입과 각종 수리까지 친절하게 상담해 드립니다.

### 행동 문구

- 전화로 바로 상담하기
- 수리 가능 여부 물어보기
- 금 매입 상담하기
- 매장 위치 확인하기

카카오톡 채널 개설 후에는 `카카오톡으로 상품 문의하기`를 주요 행동 문구로 추가한다.

## 필요한 사진

- 점포 외관: 처음 방문하는 고객이 알아볼 수 있는 모습
- 점포 내부: 실제 상담 공간과 진열 상태
- 대표 상품: 반지, 목걸이, 예물, 순금 제품
- 점주 또는 상담 모습: 친절함과 지역 점포의 인상
- 수리 작업: 작업 도구, 과정, 수리 전후 결과

사진은 실제 점포와 상품을 사용한다. 지나친 필터, 합성된 상품 이미지, 다른 점포로 오해할 수 있는 이미지는 사용하지 않는다.

## 임시 자산

로고와 실제 사진이 준비되기 전에는 교체 가능한 임시 자산을 사용할 수 있다.

- 임시 로고에는 확정되지 않은 점포명을 새로 만들지 않는다.
- 임시 사진은 레이아웃과 분위기 확인 용도로만 사용한다.
- 임시 사진을 실제 점포, 실제 판매 상품, 실제 수리 사례라고 설명하지 않는다.
- 공개 배포 전 모든 임시 자산을 점검하고 실제 자산으로 교체한다.
- 임시 자산은 파일명과 관리 데이터에서 쉽게 식별할 수 있어야 한다.
