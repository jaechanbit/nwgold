# 브랜드 방향

- 상태: 확정
- 날짜: 2026-06-18

## 배경

주요 고객은 전북 남원의 지역 주민과 선물 구매자다. 점포의 강점은 오랜 경력, 친절한 상담, 능숙한 수리 기술이다.

## 결정

브랜드의 중심 인상을 `따뜻함과 신뢰감`으로 정한다. 고급스러움은 화려한 장식보다 실제 상품과 작업 사진, 충분한 여백, 절제된 색상으로 표현한다.

## 표현 기준

- 따뜻한 아이보리 계열 배경과 어두운 본문색을 기본 방향으로 한다.
- 금색 또는 브론즈는 제한적인 강조에만 사용한다.
- 실제 점포, 인물, 상품, 수리 작업 사진을 우선한다.
- 경력과 기술은 구체적인 연도, 범위, 사례로 증명한다.

## 검토한 다른 방향

- 화려하고 고급스러운 방향: 귀금속의 인상은 강하지만 지역 고객에게 거리감을 줄 수 있다.
- 현대적이고 차가운 방향: 정돈된 인상은 있으나 점포의 친절함과 오랜 관계를 충분히 전달하기 어렵다.
